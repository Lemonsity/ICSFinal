Installing

Start by installing Python, we recommend using the latest version of Python 3

After you have Python installed, you can procceed or optionally install a text-editor, in this case, we recommend Atom


Download a lesson below to get started and open it up in your text-editor of choice.

To run a Python file:

  • open terminal or command prompt
  • change directories to where your python file (.py) is
  • run: "python filename.py" where "filename.py" is the name of your file
